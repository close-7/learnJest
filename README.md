## 前端自动化测试框架：Jest
Jest的优势：
1. 速度快
2. api简单  
3. 易配置  
4. 隔离性好  
5. 监控模式  
6. IDE整合  
7. Snapshot  
8. 多项目并行  
9. 覆盖率  
10. Mock丰富

## Jest 安装  
+ `npm install jest@24.8.0`  
+ 使用参考lesson2--通过module.export导出，require引入
+ 运行 `npm run test`
